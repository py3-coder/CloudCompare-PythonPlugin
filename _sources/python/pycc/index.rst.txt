pycc (Python)
=============


pycc is the module that wraps CloudCompare's data structures to make them
available in Python.

It also define classes and functions that allows to interact with the "CloudCompare runtime"
(that is, the currently running cloudcompare.exe which will use the python interpreter to run rou code).
